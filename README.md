# meant to be?

An interactive scrapbook web experience following an NYUAD student's study away semester in Paris. A wrong train, an unplanned destination, and a story told through layered soundscapes and flippable scrapbook pages.

## Running locally
Open `index.html` in a browser. No build step required.

## Assets
All audio and images live in `assets/` organized by scene. Audio files are `.mp3` or `.mov`. The train background is a `.gif`.
